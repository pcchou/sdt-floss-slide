Open Source?

Open Source Definition

  • http://opensource.org/osd
  • OSI 的 10 項關於 Open Source 的定義
  • Free Redistribution 自由再散步
    • 不得限制別人散布程式，也不能收授權金

  • Source Code 附上原始碼
    • 程式必須要附上原始碼，給予的方式不拘

  • Derived Works 衍生著作
    • 必須允許別人修改程式以及製作衍生著作，並散布修改後的程式。
    • 可以規定必須跟原本的程式使用相同的授權方式

  • Integrity of The Author's Source Code 原始程式碼的完整性
    • 可以要求修改後的程式慣以不同的版本號，或是要求以 patch 的方式來散布

  • No Discrimination Against Persons or Groups 不得對任何人或團體有差別待遇
    • 授權條款本身不得限制或歧視任何人禍任何團體的使用

  • No Discrimination Against Fields of Endeavor 對程式內的利用不得有差別待遇
    • 不得限制程式只可以或不可以在任何領域應用

  • Distribution of License 散布授權條款
    • 程式的授權條款必須適用到任何收到程式的使用者

  • License Must Not Be Specific to a Product 不針對特定軟體
    • 任何權利不因該軟體是不是其他產品的一部分所影響

  • License Must Not Restrict Other Software 授權條款不得影響其他軟體
    • 授權條款不得限制其他一起散布的軟體

  • License Must Be Technology-Neutral 授權條款必須技術中立
    • 授權條款允許在非網路、非圖形化、或非電腦的情況散布

Free Software?

自由軟體?

四大自由

  • 自由 0：能為任何用途自由地運行該程式
  • 自由 1：取得軟體的原始碼並學習程式的運作、修改成你想要它運作的樣子
  • 自由 2：可以自由地重新發布或複製，以幫助你的夥伴
  • 自由 3：能自由地改善程式，並將你的成果（和修改版本）釋出給大眾，嘉惠他人
